# Music Store SQL Analysis

This project contains SQL queries used to analyze the Chinook music store database.

## Questions Solved

1. Rock music listeners
2. Top spending customers by country
3. Most popular genre per country
4. Customer spending on artists

## Skills Used

- SQL Joins
- Aggregations
- Window Functions
- Common Table Expressions (CTE)
- Ranking Functions

## Dataset

Chinook Music Store Database
